# Install (Windows)

## Quick start
1. Download the latest zip from GitHub Actions artefacts (or Releases once you publish them).
2. Extract to a folder (e.g. `C:\Tools\SNOMED Lookup\`).
3. Run `SNOMED Lookup.exe`.

## First run notes
- Windows may show SmartScreen warnings for unsigned binaries. For internal use:
  - "More info" -> "Run anyway"

## Use
- Select a SNOMED CT concept ID (digits) in any app.
- Press **Ctrl+Shift+L**.
- A popup appears near your cursor.

## Logs
Logs are written to:
`%LOCALAPPDATA%\AEHRC\SNOMED Lookup\logs\app.log`
